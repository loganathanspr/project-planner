# Project Planner

A simple template-based system for planning software projects, including epics, MVPs, and issues.

## Structure

- `example-project-plan.md`: Example of a complete project plan.
- `best-practices.md`: Tips and guidelines for project planning.
- `templates/`: Markdown templates for epics, issues, and MVPs.

## Usage

1. Copy templates from the `templates/` folder to start your own project plan.
2. Fill out each section as needed for your project.
3. Refer to `example-project-plan.md` for a sample filled-out plan.

## Purpose

Helps teams organize project goals, deliverables, and tasks in a clear, repeatable format.
